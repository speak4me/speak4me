Profile Customization Tool

Create up to 8 profiles according to your needs. Then auto-generate the finished code and download it so that you only have to copy it to the Arduino. You don't know what the tool is for or how to use it? Check out our step by step guide: Step 2 - How to Use the Customization Tool

Customize your profiles

If you have already created your customizaiton with the help of our Customization Tool, you can easily import it and modify your profiles. If not you can directly start editing your expressions and use the import functionality the next time.
